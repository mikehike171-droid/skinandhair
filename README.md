# Skin & Hair Care System

A comprehensive skin and hair care management system featuring a centralized backend and a dynamic frontend.

## Project Structure

- **frontend/**: The user interface built with Next.js.
- **backend/**: The server-side logic and API services.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables in `.env` files.

3. Run the development servers:
   - Backend: `npm run dev` in the backend directory.
   - Frontend: `npm run dev` in the frontend directory.
